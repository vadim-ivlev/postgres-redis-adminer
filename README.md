# postgres-redis-adminer

Postgres + Redis + Adminer 

для проектов onlinebc onlinebc-admin и подобных.



## Запуск БД
Запустить БД

    docker-compose up -d



Postgre доступен на localhost:5432.

Аdminer - в браузере http://localhost:8080.

Параметры доступа:
- System: PostgreSQL,
- Server: db,
- Username: root,
- Password: root,
- Database: onlinebc
